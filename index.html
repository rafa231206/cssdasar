Daftar Isi

1. Pendahuluan CSS
Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mendeskripsikan presentasi dokumen yang ditulis dalam bahasa markup seperti HTML. CSS memungkinkan Anda mengontrol tampilan dan tata letak halaman web, termasuk warna, font, spasi, dan posisi elemen.

Cara Menulis CSS

Ada tiga cara utama untuk menambahkan CSS ke dokumen HTML:

  • Inline CSS: Menulis gaya langsung di dalam tag HTML menggunakan atribut style.
  • Internal CSS: Menulis gaya di dalam tag <style> di bagian <head> dokumen HTML.
  • External CSS: Menulis gaya dalam file .css terpisah dan menautkannya ke dokumen HTML menggunakan tag <link>. Ini adalah metode yang paling direkomendasikan untuk proyek besar.
<!-- Inline CSS -->
<p style="color: blue; font-size: 16px;">Teks biru.</p>

<!-- Internal CSS (di dalam <head>) -->
<style>
    h1 {
        color: brown;
    }
</style>

<!-- External CSS (di dalam <head>) -->
<link rel="stylesheet" href="style.css">

2. CSS Selector

Selector CSS digunakan untuk "menemukan" (atau memilih) elemen HTML yang ingin Anda gayakan. Berikut adalah beberapa jenis selector umum:

Universal Selector (*)

Memilih semua elemen HTML di halaman.

* {
    font-family: 'Arial';
}
Type Selector (h1, p)

Memilih semua elemen dengan nama tag tertentu.

h1 {
    color: chocolate;
}
Class Selector (.nama-kelas)

Memilih semua elemen dengan atribut class tertentu. Dapat digunakan bersama nama tag (misalnya li.sarapan).

.sarapan {
    color: darkgoldenrod;
}
ID Selector (#nama-id)

Memilih elemen dengan atribut id tertentu. ID harus unik dalam satu halaman.

#makan_siang {
    color: darkgreen;
}
Child Selector (parent > child)

Memilih elemen yang merupakan anak langsung dari elemen lain.

p > b {
    color: darkorange;
}
Descendant Selector (ancestor descendant)

Memilih elemen yang merupakan turunan dari elemen lain (tidak harus anak langsung).

p b {
    color: blue;
}
Adjacent Sibling Selector (element + element)

Memilih elemen yang langsung mengikuti elemen lain sebagai saudara.

p + h2 {
    color: deeppink;
}
General Sibling Selector (element ~ element)

Memilih semua elemen yang mengikuti elemen lain sebagai saudara (tidak harus langsung).

h2 ~ p {
    font-family: 'Times New Roman';
}

3. CSS Cascade

Cascade adalah algoritma yang menentukan bagaimana browser memutuskan nilai properti CSS mana yang akan diterapkan ketika ada beberapa aturan yang bertentangan untuk elemen yang sama. Prioritas ditentukan oleh:

  • Origin: Gaya browser, gaya pengguna, gaya penulis.
  • Specificity: Seberapa spesifik selector (ID > Class > Tag).
  • Order: Aturan yang ditulis terakhir akan menang jika spesifisitasnya sama.
  • !important: Mengesampingkan semua aturan lain kecuali aturan !important lain dengan spesifisitas lebih tinggi atau aturan !important dari gaya pengguna.
/* Order: Aturan terakhir menang */
i { color: green; }
i { color: red; } /* Ini yang akan diterapkan */

/* Specificity: ID lebih kuat dari tag */
p { font-size: 100%; }
p#pendahuluan { font-size: 75%; } /* Ini yang akan diterapkan */

/* !important: Mengesampingkan sebagian besar aturan */
b { color: pink; }
p b { color: brown !important; } /* Ini akan diterapkan */
p b { color: blue; } /* Ini diabaikan */

4. CSS Inheritance

Inheritance (pewarisan) adalah mekanisme di mana beberapa properti CSS yang diterapkan pada elemen induk secara otomatis diwarisi oleh elemen anaknya. Contoh properti yang diwarisi adalah font-family, color, dan text-align.

Anda juga bisa secara eksplisit mewarisi nilai properti dari induk menggunakan kata kunci inherit.

body {
    font-family: Arial;
    color: darkslategrey;
    padding: 10px;
}

.intro {
    border: 1px solid #665544;
    padding: inherit; /* Akan mewarisi padding dari body (10px) */
}

5. CSS Color

CSS menyediakan berbagai cara untuk menentukan warna, baik untuk teks (color) maupun latar belakang (background-color).

Nama Warna (Color Names)

Menggunakan nama warna standar seperti red, blue, darkcyan, dll.

h1 {
    color: darkcyan;
    background-color: darkkhaki;
}
Hexadecimal (HEX)

Menggunakan kode heksadesimal 6 digit (#RRGGBB) atau 3 digit (#RGB).

h2 {
    color: #ffffff; /* Putih */
    background-color: #efefef; /* Abu muda */
}
RGB (Red, Green, Blue)

Menggunakan fungsi rgb(red, green, blue) dengan nilai 0-255 untuk setiap komponen warna.

p {
    color: rgb(100, 100, 90);
    background-color: rgb(104, 228, 236);
}
Lihat file asli: 5. CSS color/warna/index.html
RGBA (Red, Green, Blue, Alpha)

Sama seperti RGB, tetapi dengan tambahan komponen alpha (transparansi) dari 0 (sepenuhnya transparan) hingga 1 (sepenuhnya buram).

h2 {
    background-color: rgba(255, 0, 119, 0.5); /* Pink 50% transparan */
}
HSL (Hue, Saturation, Lightness)

Menggunakan fungsi hsl(hue, saturation, lightness). Hue adalah sudut pada roda warna (0-360), saturation dan lightness adalah persentase (0-100%).

p.paragraf1 {
    background-color: hsl(150, 50%, 25%); /* Hijau kebiruan gelap */
}
HSLA (Hue, Saturation, Lightness, Alpha)

Sama seperti HSL, tetapi dengan tambahan komponen alpha (transparansi).

p.paragraf2 {
    background-color: hsla(150, 50%, 50%, 0.5); /* Hijau terang 50% transparan */
}

6. CSS Text

Properti CSS Text digunakan untuk mengatur tampilan teks, seperti font, ukuran, berat, perataan, dan efek lainnya.

Font Properties
  • font-family: Jenis font (misalnya 'Arial', 'Georgia').
  • font-size: Ukuran teks (misalnya 16px, 1.2em, xx-large).
  • font-weight: Ketebalan teks (misalnya normal, bold, 600).
  • font-style: Gaya teks (misalnya normal, italic).
h1 {
    font-family: Georgia;
    font-size: xx-large;
    font-weight: normal;
}
Text Alignment & Transformation
  • text-align: Perataan teks (left, right, center, justify).
  • text-transform: Mengubah kapitalisasi teks (uppercase, lowercase, capitalize).
  • text-decoration: Menambahkan atau menghapus dekorasi teks (underline, none).
h1 {
    text-transform: uppercase;
    text-align: center;
}
.credit {
    text-decoration: underline;
    text-transform: capitalize;
}
Spacing & Shadow
  • word-spacing: Jarak antar kata.
  • letter-spacing: Jarak antar huruf.
  • text-shadow: Menambahkan bayangan pada teks.
h1 {
    text-shadow: 10px 5px 5px black;
    word-spacing: 0.5em;
    letter-spacing: 5px;
}
Link States

Mengatur gaya tautan berdasarkan statusnya (belum dikunjungi, sudah dikunjungi, hover, aktif).

  • :link: Tautan yang belum dikunjungi.
  • :visited: Tautan yang sudah dikunjungi.
  • :hover: Saat kursor diarahkan ke tautan.
  • :active: Saat tautan diklik.
a.tautan:link { color: rgb(75, 20, 255); }
a.tautan:visited { color: darkgreen; }
a.tautan:hover { color: deeppink; }
a.tautan:active { color: darkcyan; }

7. CSS Box Model

Setiap elemen HTML dianggap sebagai kotak. Model kotak CSS (CSS Box Model) menjelaskan bagaimana elemen-elemen ini dirender di browser, termasuk konten, padding, border, dan margin.

Komponen Box Model
  • Content: Area tempat konten sebenarnya (teks, gambar) berada.
  • Padding: Ruang transparan di sekitar konten, di dalam border.
  • Border: Garis di sekitar padding dan konten.
  • Margin: Ruang transparan di luar border, memisahkan elemen dari elemen lain.
.content {
    width: 100px;
    height: 40px;
    padding: 10px;
    border: 20px solid red;
    margin: 20px;
}

Properti Box Sizing

Properti box-sizing menentukan bagaimana width dan height elemen dihitung:

  • content-box (default): width dan height hanya mencakup konten. Padding dan border ditambahkan di luar ukuran ini.
  • border-box: width dan height mencakup konten, padding, dan border. Ini lebih intuitif untuk tata letak.
* {
    box-sizing: border-box;
}
Width & Height

Mengatur lebar dan tinggi elemen. Bisa dalam piksel (px), persentase (%), atau unit lainnya.

  • min-width / max-width: Lebar minimum/maksimum.
  • min-height / max-height: Tinggi minimum/maksimum.
  • fit-content: Menyesuaikan ukuran dengan konten.
.box1 {
    height: 100px;
    width: 300px;
}
td.deskripsi {
    min-width: 450px;
    max-width: 500px;
}
div.box4 {
    width: fit-content;
    height: fit-content;
}

Overflow

Mengontrol apa yang terjadi ketika konten elemen terlalu besar untuk muat di dalam kotak elemen.

  • visible (default): Konten akan meluap keluar dari kotak.
  • hidden: Konten yang meluap akan dipotong dan disembunyikan.
  • scroll: Scrollbar akan ditambahkan untuk melihat konten yang meluap.
  • auto: Scrollbar akan ditambahkan hanya jika diperlukan.
p.satu {
    height: 100px;
    width: 200px;
    overflow: hidden;
}
p.dua {
    height: 100px;
    width: 200px;
    overflow: scroll;
}

10. CSS Margin

Margin adalah ruang di luar border elemen, digunakan untuk memisahkan elemen dari elemen lain.

11. Display Element

Properti display adalah salah satu properti CSS yang paling penting, karena mengontrol bagaimana elemen dirender dan berinteraksi dengan elemen lain dalam tata letak.

display: inline

Elemen ditampilkan sebaris dengan teks, dan tidak dapat memiliki lebar atau tinggi yang ditentukan. Contoh: <span>, <a>.

span.inline {
    display: inline;
    width: 100px; /* Tidak akan berpengaruh */
}
display: block

Elemen dimulai pada baris baru dan mengambil lebar penuh yang tersedia. Dapat memiliki lebar dan tinggi yang ditentukan. Contoh: <div>, <p>, <h1>.

span.block {
    display: block;
    width: 100px; /* Akan berpengaruh */
}
display: inline-block

Elemen ditampilkan sebaris, tetapi dapat memiliki lebar dan tinggi yang ditentukan. Berguna untuk membuat elemen seperti tombol atau kartu yang sejajar tetapi memiliki dimensi.

span.inline-block {
    display: inline-block;
    width: 100px;
    height: 100px;
}
display: flex (Flexbox)

Mengubah elemen menjadi kontainer flex, memungkinkan penataan item anak secara fleksibel dalam satu dimensi (baris atau kolom).

.flex-container {
    display: flex;
    flex-wrap: wrap;
    justify-content: center;
}

12. List Style

Properti list-style digunakan untuk mengatur tampilan penanda item daftar (bullet atau angka).

list-style-type

Menentukan jenis penanda:

  • Untuk <ol>: decimal, lower-roman, upper-roman, lower-latin, upper-latin, lower-greek.
  • Untuk <ul>: disc (default), circle, square, none.
ol { list-style-type: decimal; }
ul.singkat { list-style-type: circle; }
list-style-position

Menentukan posisi penanda relatif terhadap item daftar:

  • inside: Penanda berada di dalam kotak konten item daftar.
  • outside (default): Penanda berada di luar kotak konten item daftar.
ul li { list-style-position: inside; }
list-style (Shorthand)

Shorthand untuk mengatur list-style-type, list-style-position, dan list-style-image.

ol.singkat { list-style: lower-greek inside; }

16. CSS Position

Properti position digunakan untuk mengontrol penempatan elemen di halaman web.

position: static (Default)

Elemen dirender dalam urutan normal dokumen. Properti top, right, bottom, left tidak berpengaruh.

position: relative

Elemen diposisikan relatif terhadap posisi normalnya. Menggunakan top, right, bottom, left akan menggeser elemen dari posisi aslinya, tetapi ruang yang ditempati elemen tetap dipertahankan dalam aliran dokumen.

p {
    position: relative;
    top: 100px;
    background-color: aqua;
}
.posisi-relative {
    position: relative;
    left: 50px;
    top: 50px;
    background-color: coral;
}

position: absolute

Jika tidak ada induk dengan posisi non-static, elemen akan diposisikan relatif terhadap <html>. Elemen yang diposisikan absolut dihapus dari aliran normal dokumen.

    background-color: coral;

21. Grid Layout

CSS Grid Layout adalah sistem tata letak dua dimensi yang sangat kuat, memungkinkan Anda mendesain tata letak kompleks dengan baris dan kolom.

  • display: grid: Mengubah elemen menjadi kontainer grid.
  • grid-template-columns: Menentukan jumlah dan lebar kolom.
  • grid-template-rows: Menentukan jumlah dan tinggi baris.
  • gap: Jarak antar baris dan kolom.
  • grid-column-start / grid-column-end: Mengatur item untuk membentang di beberapa kolom.
  • grid-row-start / grid-row-end: Mengatur item untuk membentang di beberapa baris.
.grid-container {
    display: grid;
    grid-template-columns: 10% 30% auto;
    grid-template-rows: 100px 200px auto;
    gap: 10px 10px;
}
.item1 {
    grid-column-start: 1;
    grid-column-end: 3;
}
.item5 {
    grid-row-start: 3;
    grid-row-end: 5;
}

23. Responsive Layout

Tata letak responsif adalah desain web yang memungkinkan tata letak halaman beradaptasi dengan berbagai ukuran layar dan perangkat. Ini sering dicapai dengan kombinasi unit relatif (persentase), media queries, dan Flexbox/Grid.

Media Queries

Media queries memungkinkan Anda menerapkan gaya CSS yang berbeda berdasarkan karakteristik perangkat, seperti lebar layar.

/* Gaya default untuk semua layar */
[class*="col-"] {
    width: 100%; /* Default: semua kolom 100% lebar (untuk mobile) */
    float: left;
    padding: 10px;
}

/* Gaya untuk layar dengan lebar minimal 768px (tablet ke atas) */
@media only screen and (min-width:768px) {
    .col-1 { width: 8.33%; }
    .col-2 { width: 16.66%; }
    /* ... dan seterusnya untuk kolom lainnya */
    .footer {
        position: fixed;
        bottom: 0px;
        left: 0px;
    }
    .menu li {
        width: 20%;
        display: inline-block;
